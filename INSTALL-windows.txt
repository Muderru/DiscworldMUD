1) Uncompress the package so that the folders like bin and lib
   are in c:\dw
2) Use wordpad to edit c:\dw\bin\dw.cfg.win32 and change the mudname
3) Use wordpad to edit c:\dw\lib\secure\master.c and change "cratylus" to your admin's name.
4) Double-click on c:\dw\win32\mud.bat
5) telnet to localhost 1701

Enjoy!

If you run into trouble, feel free to post on the Discworld
section of http://lpmuds.net/forum
